## 图片规格建议

### 格式
- **推荐**: JPG（照片类）、PNG（需要透明背景）
- **压缩**: 使用 TinyPNG 或类似工具压缩，保持质量同时减小文件大小

### 尺寸要求
- **Hero 卡片**: 800×600px，横版 4:3 比例
- **产品图片**: 600×400px，横版 3:2 比例
- **博客封面**: 600×340px，横版约 16:9 比例
- **头像**: 80×80px，正方形

### 内容建议
- **产品图**: 白色或浅色背景，突出产品细节
- **Hero 图**: 展示产品应用场景，有品牌感
- **头像**: 专业商务风格，正脸清晰
